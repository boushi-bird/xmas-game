# xmas-game

[![CircleCI](https://circleci.com/gh/boushi-bird/xmas-game.svg?style=svg)](https://circleci.com/gh/boushi-bird/xmas-game)

## セットアップ
```
npm install
```

### ローカルでバッグ
```shell
npm run serve
# open http://localhost:8080
```

### ビルド
```shell
npm run build
```

### 構文チェック
```shell
npm run lint
```

### E2Eテスト
```shell
npm run test:e2e
```

### ユニットテスト
```shell
npm run test:unit
```

### デプロイ
```shell
npm run deploy
```
